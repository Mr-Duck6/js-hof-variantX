utils.js  Функції вищого порядку (pipe, compose) для створення функціональних конвеєрів.
validators.js  Бібліотека атомарних валідаторів. Використовує каррінг для гнучкого налаштування правил.
validationEngine.js  Ядро системи. Асинхронно агрегує результати перевірок за допомогою Promise.all та методів масивів (filter, map).
formRules.js  Декларативна конфігурація правил для конкретних полів інтерфейсу.
main.js  Вхідна точка. Обробляє події та виконує маніпуляції з DOM.
